Ova demo-stranica je izrađena za kolegij razvoj interakcijskih sučelja – pokazuje osnovne HTML elemente, moderne CSS stilove (flex i grid), animacije i responzivni dizajn.

Istaknute vještine

HTML osnove

Razumijem strukturu web stranice i pravilno koristim osnovne HTML elemente poput naslova, slika i linkova.

CSS stilovi

Primjenjujem boje, margine, tipografiju i raspored elemenata kako bi stranica izgledala uredno i moderno

Animacije i prijelazi

Koristim jednostavne hover efekte i tranzicije da stranica bolje izgleda.

Responzivni dizajn

izgled stranice različitim uređajima pomoću media queryja i fleksibilnog rasporeda.

Alati koje koristim

Radim u alatima poput Visual Studio Codea, Figme i Photoshopa za izradu i uređivanje dizajna.

Kreativnost i dizajn

Volim kombinirati boje, fontove i slike kako bi stranice bile privlačne i lako čitljive.
